## Features

### Users
- Can order a product,
- Can pay for an order,
- see his/her profile and update,
- see his/ her orders and payment status,
- see transactionId,
- etc.

### Admin 
- can add a product,
- can remove a product,
- can make another one an admin,
- deliver orders.
- etc
